HTTP METHODS: 
1) Get
2) Post
3) Put
4) Patch
5) Delete

Get: When you want to get some data from the server.
Post: When you want to send and mutate some data in the server. For eg:sign up/login
Put: When you want to upload any image or other file.
Patch: When you want to rename some data. Like, changing the password in any website
Delete: When you want to delete some data.
